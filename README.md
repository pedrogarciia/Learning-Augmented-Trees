# Learning-Augmented-Trees
Learning-Augmented Data Structures, com foco em árvores de busca.  O objetivo é demonstrar, de forma prática, como previsões (simulando o uso de Machine Learning) podem ser utilizadas para otimizar a organização de uma árvore binária de busca, aproximando elementos mais frequentemente acessados da raiz e reduzindo o custo médio das operações.
